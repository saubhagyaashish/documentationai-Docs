---
title: Creating a New Project
description: How to create a project in Saubhagya Ashish.
---

## Steps

<Steps>
  <Step title="Open Dashboard">
    ````
    Go to https://saubhagyaashish.com and log in to your account.
    ````
  </Step>
  <Step title="Create Project">
    ````
    Click on "New Project", enter project name and description, then click "Create".
    ````
  </Step>
  <Step title="Start Documenting">
    ````
    Select your new project to add sections and write documentation.
    ````
  </Step>
</Steps>
