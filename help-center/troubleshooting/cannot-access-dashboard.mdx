title: Cannot Access Dashboard
description: Fix dashboard access issues in Saubhagya Ashish.
---

## Symptoms

- Dashboard won't load
- See a "403 Forbidden" message

## Causes

- Not logged in
- Account permissions issue

## Quick Fix

<Steps>
  <Step title="Check login">
    ````
    Make sure you are signed in at https://saubhagyaashish.com/login
    ````
  </Step>
  <Step title="Check permissions">
    ````
    Only invited users can access private project dashboards. Contact admin if needed.
    ````
  </Step>
</Steps>
