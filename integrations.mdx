## Overview

Saubhagya Ashish supports seamless integrations with popular third-party tools. You can set up webhooks for real-time notifications, access the API for custom connections, and export your documentation to various formats. These features streamline your workflow and keep your team in sync.

<Callout kind="tip">
  Start with supported app integrations for quick setup, then explore webhooks and API for advanced use cases.
</Callout>

## Setting Up Webhooks

Webhooks enable real-time notifications when docs update. Receive payloads at your endpoint for custom automations.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to `https://dashboard.example.com/settings/webhooks` and click "New Webhook".
  </Step>
  <Step title="Configure URL" icon="link">
    Enter your endpoint URL, such as `https://your-webhook-url.com/saubhagya`.
  </Step>
  <Step title="Select Events" icon="zap">
    Choose events like `doc.updated` or `page.published`.
  </Step>
  <Step title="Test and Save" icon="check-circle">
    Send a test payload and save the webhook.
  </Step>
</Steps>

Here is an example webhook payload:

```json
{
  "event": "doc.updated",
  "doc_id": "doc_123abc",
  "timestamp": "2024-10-15T10:30:00Z",
  "user_id": "user_456def"
}
```

<CodeGroup tabs="cURL,JavaScript">
  ```bash
  curl -X POST https://your-webhook-url.com/saubhagya \
    -H "Content-Type: application/json" \
    -d '{
      "event": "doc.updated",
      "doc_id": "doc_123abc"
    }'
  ```
  ```javascript
  fetch('https://your-webhook-url.com/saubhagya', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      event: 'doc.updated',
      doc_id: 'doc_123abc'
    })
  });
  ```
</CodeGroup>

## API Access for Custom Connections

Build custom integrations using the REST API at `https://api.example.com/v1`. Authenticate with your API key.

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField query="doc_id" param-type="string" required="false">
  Filter by specific document ID.
</ParamField>

Example request to fetch docs:

<Request tabs="JavaScript,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/docs', {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  });
  const docs = await response.json();
  ```
  ```bash
  curl "https://api.example.com/v1/docs" \
    -H "Authorization: Bearer YOUR_API_KEY"
  ```
</Request>

<Response tabs="200">
```json
{
  "docs": [
    {
      "id": "doc_123abc",
      "title": "Project Overview",
      "updated_at": "2024-10-15T10:30:00Z"
    }
  ]
}
```
</Response>

## Exporting Documentation

Export your docs to external formats for sharing or archiving. Use the dashboard or API.

<Tabs>
  <Tab title="PDF" icon="file">
    Generate PDF exports from the dashboard.
    
    ```bash
    curl "https://api.example.com/v1/docs/doc_123abc/export?format=pdf" \
      -H "Authorization: Bearer YOUR_API_KEY" \
      --output docs.pdf
    ```
  </Tab>
  <Tab title="Markdown" icon="file-text">
    Export as Markdown for GitHub or static sites.
    
    ```bash
    curl "https://api.example.com/v1/docs/export?format=md&doc_id=doc_123abc" \
      -H "Authorization: Bearer YOUR_API_KEY" \
      --output docs.md
    ```
  </Tab>
  <Tab title="HTML" icon="code">
    Get clean HTML for embedding.
    
    ```bash
    curl "https://api.example.com/v1/docs/export?format=html" \
      -H "Authorization: Bearer YOUR_API_KEY" \
      --output docs.html
    ```
  </Tab>
</Tabs>
